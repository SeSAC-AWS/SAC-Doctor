  • Router 구축

    Router을 통한 NAT,
    DHCP을 통한 IP 할당

  • ESXi 구축 및 VM 설치

    ESXi구축을 통한 백업 파일 보관,
    VM 설치를 통한 각 서버 구축

  • WebServer / DNS DB 구축

    VM 상에 WebServer 구축 및 홈페이지 제작,
    DNS DB 구축을 통한 홈페이지 도메인 관리

  • SMB / NFS / FTP / Webshare 구축

    내부 파일 공유 시스템을 통한 파일 공유

  • SLB / NMS 구축

    로드밸런싱을 통한 홈페이지 가용성 확보,
    NMS을 활용한 네트워크 모니터링
